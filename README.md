# Genie Job Quest

AI-powered job search assistant built using Lovable, n8n, OpenAI and Apify.

## Features

- Resume upload
- Job matching
- Job scoring
- Tailored resume generation
- Tailored cover letter generation

## Live Demo

https://genie-job-quest.lovable.app

## Tech Stack

- Lovable
- n8n
- OpenAI
- Apify


## Architecture

User → Lovable UI → n8n Workflow → AI Processing → Results
